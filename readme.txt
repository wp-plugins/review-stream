=== Review Stream ===
Contributors: gradeus
Tags: reviews, reputation, seo
Tested up to: 4.1.1

Stream your latest and greatest reviews from around the Web to your Wordpress site and display them with SEO-friendly rich-snippet markup.

== Description ==

To use this plugin, you must:

* have a Grade.us account with review monitoring enabled
* be approved for API access and know your API token
* add the [reviewstream] shortcode to your Wordpress site
